# ztl
zsh with tmux logging some command.

## Demo

![result](https://github.com/nnao45/naoGifRepo/blob/master/ztl.gif)

## Install

### Zplug

write zshrc

```
$ zplug "nnao45/ztl", use:'src/_*' 
```

### Source Install

Download Repo

```
$ mkdir ${HOME}/.ztl
$ git clone https://github.com/nnao45/ztl.git ${HOME}/.ztl/
```

write zshrc

```
$ echo source ${HOME}/.ztl/src/_*
```
